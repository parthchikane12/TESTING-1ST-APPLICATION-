# ADUKUNDA – Mobile Application Testing Project

## 📌 Project Overview

This repository contains the **manual testing documentation** for the **ADUKUNDA mobile application** tested as part of software testing practice with **Test Yantra**.
The objective of this project is to validate the functionality, usability, and reliability of the application through structured test cases and bug reporting.

---

## 🎯 Project Objectives

* To perform **manual testing** on the ADUKUNDA mobile application.
* To identify functional defects in the application.
* To document **test cases, bug reports, and test plan**.
* To improve software quality through systematic testing.

## 🧪 Testing Scope

The testing scope includes verification of the following features:

* Application Launch
* User Login and Authentication
* OTP Verification
* Navigation and Menu Options
* Profile Management
* Search Functionality
* Notifications
* Settings
* Logout Functionality
* Error Handling

---

## 🛠 Testing Strategy

The testing process follows **Manual Testing methodology** with the following testing types:

* Functional Testing
* UI Testing
* Usability Testing
* Error Handling Testing
* Basic Performance Observation

Test cases were executed on the mobile application and defects were recorded in the bug report sheet.
